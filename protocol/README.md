rpc 协议设计

格式如下


**请求报文格式**
```
--------------------------------------------------------------
| magic  | version |  type  | Host | Sequence | Encode | Body|
--------------------------------------------------------------
```


**响应报文格式**
```
---------------------------------------------------------
| status  | version |  type  |  Sequence | Encode | Body|
---------------------------------------------------------
```

默认 header 用 gob 编码，body 用 pb 编码
